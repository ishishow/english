## 16.1 重要エラー：exist の正しい使い方

```
❌ Bad: "it's not exist"
✅ Good: "it doesn't exist"

❌ Bad: "it do not exist"
✅ Good: "it does not exist"

❌ Bad: "if it exists, I was nervous"
✅ Good: "if it existed, I would be nervous"
```

**ルール**: exist は動詞なので、否定には does not / do not を使う

| 文型 | 正しい形 |
|------|----------|
| 現在形 | It exists / It doesn't exist |
| 過去形 | It existed / It didn't exist |
| 仮定法 | If it existed, I would... |

---

## 16.2 重要エラー：仮定法の使い方

```
❌ Bad: "if it exists, I was nervous"
✅ Good: "if it existed, I would be nervous"

❌ Bad: "if you chose past, you can change history"
✅ Good: "if you chose the past, you could change history"
```

**仮定法過去**: 現実ではないことを仮定する

| パターン | 例文 |
|----------|------|
| If + 過去形, would + 動詞 | If I had a time machine, I would go to the future |
| If + 過去形, could + 動詞 | If you went to the past, you could change history |

---

## 16.3 重要エラー：hide の過去分詞

```
❌ Bad: "that was hide by someone"
✅ Good: "that was hidden by someone"
```

| 動詞 | 過去形 | 過去分詞 |
|------|--------|----------|
| hide | hid | hidden |
| ride | rode | ridden |
| write | wrote | written |

## 16.7 新出イディオム・表現

### "as far as I know"
**意味**: 私の知る限り
```
例: As far as I know, time travel is not possible.
    私の知る限り、タイムトラベルは不可能です。
```

### "learning never ends" / "learning is a continuous process"
**意味**: 学びに終わりはない
```
例: Learning is a continuous process. You can always improve.
    学びは継続的なプロセスです。常に向上できます。
```

### "gain confidence"
**意味**: 自信を得る
```
例: I gained confidence to speak English.
    英語を話す自信がついた。
```

## 16.11 練習問題

### A. 正しい形に直せ

1. Time machines is not exist.
2. If it exists, I was scared.
3. The truth was hide by someone.
4. I studied to study English.

### B. 仮定法で書き換えよ

1. I don't have a time machine. I can't go to the future.
   → If I ___, I ___.
2. Time travel is not possible. We can't change history.
   → If time travel ___, we ___.

### C. 日本語を英語に

1. タイムマシンがあったら、過去に行くだろう
2. 学びに終わりはない
3. 英語を話す自信がついた
4. 転職するために英語を勉強している

## 16.12 練習問題 解答

### A. 正しい形に直せ
1. Time machines **do not exist**.
2. If it **existed**, I **would be** scared.
3. The truth was **hidden** by someone.
4. I **started studying** English. / I've **been studying** English.

### B. 仮定法で書き換えよ
1. If I **had a time machine**, I **could go to the future**.
2. If time travel **were possible**, we **could change history**.

### C. 日本語を英語に
1. If I had a time machine, I would go to the past.
2. Learning never ends. / Learning is a continuous process.
3. I gained confidence in speaking English.
4. I'm studying English to change my job.

## 16.13 総合評価

| 項目 | 評価 | コメント |
|------|------|---------|
| exist の使い方 | C | be動詞と混同している |
| 仮定法 | C | If + 過去形, would + 原形 の練習が必要 |
| 過去分詞 | C | hidden など不規則動詞の復習が必要 |
| 留学経験の説明 | B+ | 動機や経験を説明できている |
| 表現力 | B | 難しいトピックにも挑戦している |
