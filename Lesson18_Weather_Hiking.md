## 18.1 重要エラー：live vs leave の継続的な混同

```
❌ Bad: "I can't leave 50 temperature"
✅ Good: "I can't live in 50 degree weather" / "I can't survive in 50 degrees"

❌ Bad: "I can't leave"
✅ Good: "I can't live there"
```

**再確認**:
- **live** /lɪv/ = 住む、生きる、生存する
- **leave** /liːv/ = 去る、出発する

## 18.2 重要エラー：temperature の表現

```
❌ Bad: "50 temperature"
✅ Good: "50 degrees" / "a temperature of 50 degrees"

❌ Bad: "cold temperature a degree"
✅ Good: "cold temperatures"
```

### 気温を表現する

| 日本語 | 英語 |
|--------|------|
| 50度 | 50 degrees (Celsius) |
| マイナス10度 | minus 10 degrees / -10 degrees |
| 気温が上がる | the temperature rises / goes up |
| 気温が下がる | the temperature drops / goes down |

## 18.3 重要エラー：形容詞の使い方

```
❌ Bad: "it's low" (緑の割合について)
✅ Good: "that's a low percentage" / "that's not much"

❌ Bad: "it's good objective"
✅ Good: "that's a good adjective"
```

**注意**: objective（目的）と adjective（形容詞）を混同

| 単語 | 意味 |
|------|------|
| objective | 目的、目標 |
| adjective | 形容詞 |

## 18.7 新出イディオム・表現

### "do justice to ~"
**意味**: 〜を正当に評価する、〜の良さを伝える
```
例: The picture is not doing justice with the real beauty.
    写真は実際の美しさを伝えきれていない。
```

### "be into ~"
**意味**: 〜に夢中である、〜が好きである
```
例: My friend is into hiking.
    私の友達はハイキングに夢中だ。
```

### "go for ~"
**意味**: 〜を選ぶ
```
例: If there were only two seasons, I would go for summer.
    2つの季節しかなければ、夏を選ぶだろう。
```

## 18.11 練習問題

### A. live と leave を正しく使い分けよ

1. I can't ___ in such hot weather.
2. I will ___ Japan next month.
3. Where do you ___?
4. I want to ___ early tomorrow.

### B. 正しい形に直せ

1. It's 50 temperature today.
2. I like go outside.
3. Let's go to hike this weekend.
4. That's a good objective. (「形容詞」と言いたい)

### C. 日本語を英語に

1. 気温が50度まで上がる
2. 日本では元日に初日の出を見ます
3. 私の友達はハイキングに夢中です
4. 写真では実際の美しさが伝わらない

---

## 18.12 練習問題 解答

### A. live と leave を正しく使い分けよ
1. I can't **live** in such hot weather.
2. I will **leave** Japan next month.
3. Where do you **live**?
4. I want to **leave** early tomorrow.

### B. 正しい形に直せ
1. It's **50 degrees** today.
2. I like **going** outside.
3. Let's **go hiking** this weekend.
4. That's a good **adjective**.

### C. 日本語を英語に
1. The temperature rises/goes up to 50 degrees.
2. In Japan, we watch the first sunrise of the year on New Year's Day.
3. My friend is into hiking.
4. The picture doesn't do justice to the real beauty.
