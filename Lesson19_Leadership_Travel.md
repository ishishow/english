## 19.1 重要エラー：used to の使い方

```
❌ Bad: "I used to being, I used to working software as a software engineer"
✅ Good: "I'm used to working as a software engineer"
      または "I used to work as a software engineer"
```

### used to の2つの意味

| 形 | 意味 | 例文 |
|----|------|------|
| used to + 動詞の原形 | 以前は〜していた（今はしていない） | I used to smoke. |
| be used to + ~ing | 〜に慣れている | I'm used to waking up early. |
| get used to + ~ing | 〜に慣れる | I'm getting used to the new job. |

```
例文:
- I used to play soccer. (以前サッカーをしていた)
- I'm used to working long hours. (長時間労働に慣れている)
- I'm getting used to speaking English. (英語を話すことに慣れてきた)
```

## 19.12 練習問題

### A. used to を正しく使い分けよ

1. I ___ smoke, but I quit last year. (以前吸っていた)
2. I'm ___ waking up early. (慣れている)
3. I'm getting ___ the new environment. (慣れてきた)
4. He ___ live in Tokyo. (以前住んでいた)

### B. 正しい形に直せ

1. I went to Taiwanese last year.
2. I want to have another excuse.
3. My weakness point is shy.
4. Compare with Asia, Europe is safe.

### C. 日本語を英語に

1. 転職したいです
2. 3人の部下を管理しています
3. 友達がスリに遭いました
4. シャイを克服したい

## 19.13 練習問題 解答

### A. used to を正しく使い分けよ
1. I **used to** smoke, but I quit last year.
2. I'm **used to** waking up early.
3. I'm getting **used to** the new environment.
4. He **used to** live in Tokyo.

### B. 正しい形に直せ
1. I went to **Taiwan** last year.
2. I want to **gain other skills**.
3. My weakness is **shyness**. / I'm **too shy**.
4. **Compared to/with** Asia, Europe is safe.

### C. 日本語を英語に
1. I want to change jobs. / I want to switch companies.
2. I manage three people. / I have three people under me.
3. My friend got pickpocketed.
4. I want to overcome my shyness.
