## 23.2 重要エラー：受動態

```
❌ Bad: "window glass may broken"
✅ Good: "window glass may break" / "window glass may be broken"

❌ Bad: "a big earthquake cause breaking buildings"
✅ Good: "a big earthquake can cause buildings to collapse"
```

### 受動態の形

| 能動態 | 受動態 |
|--------|--------|
| break | be broken |
| damage | be damaged |
| destroy | be destroyed |

---

## 23.3 重要エラー：be used to の使い方

```
❌ Bad: "we used to living with earthquake"
✅ Good: "we are used to living with earthquakes"
```

### パターン確認

| 形 | 意味 | 例文 |
|----|------|------|
| be used to + ~ing | 〜に慣れている | I'm used to living here. |
| get used to + ~ing | 〜に慣れる | I'm getting used to it. |
| used to + 原形 | 以前は〜だった | I used to live there. |

## 23.5 重要フレーズ：地震対策

### 地震発生時の行動

| 日本語 | 英語 |
|--------|------|
| 机の下に隠れる | hide under the desk |
| 窓から離れる | move away from windows |
| ドアを開けておく | leave the door open |
| 落ち着く | stay calm |
| 協力する | cooperate with each other |

```
例文:
- When an earthquake happens, move under the desk.
  地震が起きたら、机の下に移動してください。

- Move away from glass windows because they may break.
  ガラス窓は割れる可能性があるので離れてください。

- It's a good idea to leave the door open.
  ドアを開けておくのが良いです。
```

## 23.9 新出イディオム・表現

### "inflicted by"
**意味**: 〜の被害を受ける
```
例: We are not inflicted by natural disasters.
    私たちは自然災害の被害を受けていません。
```

### "running out of time"
**意味**: 時間がなくなる
```
例: We are running out of time.
    時間がなくなってきました。
```

### "seldom"
**意味**: めったに〜ない
```
例: Flooding seldom happens here.
    ここではめったに洪水は起きません。
```

## 23.11 練習問題

### A. glass と grass の発音練習
1. The window ___ may break during an earthquake.
2. The children played on the ___.

### B. be used to を正しく使え
1. Japanese people ___ (are used to / used to) living with earthquakes.
2. I ___ (am used to / used to) smoke, but I quit.

### C. 日本語を英語に
1. 地震が起きたら机の下に隠れてください
2. 日本は厳しい建築基準があります
3. ソーシャルメディアは依存性があります

---

## 23.12 練習問題 解答

### A. glass と grass
1. The window **glass** may break during an earthquake.
2. The children played on the **grass**.

### B. be used to
1. Japanese people **are used to** living with earthquakes.
2. I **used to** smoke, but I quit.

### C. 日本語を英語に
1. When an earthquake happens, hide under the desk.
2. Japan has strict building standards.
3. Social media is addictive.
