## 26.1 重要エラー：単数・複数形

```
❌ Bad: "I have been to other country"
✅ Good: "I have been to other countries"

❌ Bad: "each area, each there are a lot of kind of ozoni"
✅ Good: "each area has its own kind of ozoni"
```

### 先生が教えてくれた自然な表現

```
Each area has its own kind of ozoni.
各地域には独自のお雑煮があります。
```

## 26.3 重要エラー：現在完了進行形

```
❌ Bad: "recently, I don't read manga"
✅ Good: "recently, I haven't been reading manga"
```

### 現在完了進行形の使い方

| 形 | 意味 | 例文 |
|----|------|------|
| have been + ~ing | 最近ずっと〜している/していない | I haven't been reading manga recently. |

```
例文:
- I haven't been studying English recently.
  最近英語を勉強していない。

- I've been working a lot lately.
  最近たくさん働いている。
```

## 26.5 重要フレーズ：日本の郷土料理

### お雑煮を説明する

```
Ozoni is a traditional New Year soup in Japan.
お雑煮は日本の伝統的な正月の汁物です。

It contains mochi (rice cakes) and local vegetables.
餅と地元の野菜が入っています。

Each area has its own kind of ozoni.
各地域には独自のお雑煮があります。

Some places add pumpkin and noodles.
かぼちゃや麺を入れる地域もあります。
```

### お正月について

| 日本語 | 英語 |
|--------|------|
| 正月 | New Year |
| 正月期間 | Oshogatsu (New Year week) |
| 1月1日から6日まで | from January 1st to the 6th |
| 祝日週間 | holiday week |

### コロンビア料理

| 日本語 | 英語 |
|--------|------|
| アヒアコ | Ajiaco (Colombian soup) |
| サンコチョ | Sancocho (meat and vegetable soup) |

  エンパナーダは餃子に似ていますが揚げています。

## 26.9 新出イディオム・表現

### "up to date"
**意味**: 最新の状態
```
例: I'm up to date with the manga.
    漫画は最新話まで読んでいます。
```

### "each area has its own ~"
**意味**: 各地域には独自の〜がある
```
例: Each area has its own kind of ozoni.
    各地域には独自のお雑煮があります。
```

### "I feel the same"
**意味**: 同感です
```
例: I'm getting hungry too. I feel the same.
    私もお腹が空いてきました。同感です。
```

## 26.12 練習問題 解答

### A. 正しい形
1. I have been to other **countries**.
2. Recently, I **haven't been reading** manga.
3. **Each area has its own kind of** ozoni.

### B. 発音練習
1. vegetables - /ˈvedʒtəblz/
2. coriander - /ˌkɔːriˈændər/
3. traditional - /trəˈdɪʃənl/

### C. 日本語を英語に
1. Each area has its own kind of ozoni.
2. I haven't been reading manga recently.
3. It takes 1.5 hours by Shinkansen.
