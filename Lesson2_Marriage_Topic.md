# 今回の重要な修正ポイント

## 1. 超重要！ Married vs Money の発音

今回のレッスンで最も危険だった間違い：

```
⚠️ 音声認識で "married" が "money" に誤変換されていました！

❌ "I want to get money in the future"
   → お金が欲しい（意味が全く違う！）

✅ "I want to get married in the future"
   → 結婚したい

❌ "It's difficult to get money"
✅ "It's difficult to get married"
```

### 発音のコツ
| 単語 | 発音 | 注意点 |
|------|------|--------|
| Married | /ˈmærɪd/ | 「マリッド」- Rをはっきり |
| Money | /ˈmʌni/ | 「マニー」 |

**練習文**:
- I want to get **married** someday.
- Do you think love is necessary for a good **marriage**?

## 2. Live vs Leave の混同

```
⚠️ これも危険な間違い！

❌ "Many friends advise them to leave each other"
   → 別れるよう助言する

✅ "Many friends advise them to live with each other"
   → 一緒に住むよう助言する
```

### 発音の違い
| 単語 | 発音 | 意味 |
|------|------|------|
| Live | /lɪv/ | 住む、生きる |
| Leave | /liːv/ | 去る、離れる |

**ポイント**: Live は短い「イ」、Leave は長い「イー」

---

## 3. 「思いつかない」の表現

### Original:
```
❌ "I don't come up with some any benefit"
```

### Correction:
```
✅ "I can't think of any benefits"
✅ "Nothing comes to mind"
✅ "I can't come up with any benefits"
```

### 解説
- "come up with" は**肯定文**や**疑問文**で使うのが自然
- 否定文では "can't think of" の方が自然
- "some any" は重複 → どちらか一方だけ使う

---

## 4. 「人生の転機」の表現

### Original:
```
❌ "It's a changing life period"
❌ "It's a changing life time"
```

### Correction:
```
✅ "It's a life-changing event"
   （人生を変える出来事だ）

✅ "It's a major turning point in life"
   （人生の大きな転機だ）

✅ "It's a big milestone"
   （大きな節目だ）
```

### 解説
- "life-changing" はハイフンでつなげて**形容詞**として使う
- "changing life" だと「変化している人生」という不自然な意味に

## 5. 信頼を築く/失う

### Original:
```
❌ "Marriage is a creating trust to my partner"
❌ "I will lose my trust"
```

### Correction:
```
✅ "Marriage helps build trust with my partner"
   （結婚はパートナーとの信頼を築くのに役立つ）

✅ "She will lose trust in me"
   （彼女は私への信頼を失うだろう）

✅ "I will lose her trust"
   （私は彼女の信頼を失うだろう）
```

### 信頼に関する重要表現
| 表現 | 意味 |
|------|------|
| build trust | 信頼を築く |
| lose trust in someone | 〜への信頼を失う |
| gain someone's trust | 〜の信頼を得る |
| break trust | 信頼を壊す |
| trust issues | 信頼の問題 |

## 6. 「〜するべき」の表現

### Original:
```
❌ "We should do that"（曖昧）
```

### より自然な表現:
```
✅ "It's common to do that"
   （そうするのが一般的です）

✅ "It's expected that couples do that"
   （カップルがそうすることは期待されています）

✅ "That's the norm in Japan"
   （日本ではそれが普通です）
```

---

## 7. 「世間体を気にする」

### Original:
```
❌ "We care about some people's eyesight"
   → 「人々の視力を気にする」という意味に！
```

### Correction:
```
✅ "We care about what people think"
   （人がどう思うかを気にする）

✅ "We care about social reputation"
   （社会的評判を気にする）

✅ "We worry about being judged"
   （評価されることを心配する）
```

### 関連イディオム
| 表現 | 意味 |
|------|------|
| Keep up appearances | 体裁を保つ |
| Save face | 面目を保つ |
| What will people say? | 人がなんて言うか |
| Social stigma | 社会的偏見 |

---

## 8. 離婚率の表現

### Original:
```
❌ "The divorce percentage is 30%"
```

### Correction:
```
✅ "The divorce rate is 30%"
   （離婚率は30%です）

✅ "30% of marriages end in divorce"
   （結婚の30%が離婚に終わる）
```

### 統計を語る表現
| 表現 | 意味 |
|------|------|
| The divorce rate | 離婚率 |
| The marriage rate | 結婚率 |
| end in divorce | 離婚に終わる |
| file for divorce | 離婚を申請する |

---

## 9. 「基準が高すぎる」

### Original:
```
❌ "Standard is too high"
```

### Correction:
```
✅ "Standards are too high"
   （基準が高すぎる）※複数形！

✅ "People have unrealistic expectations"
   （人々は非現実的な期待を持っている）

✅ "The bar is set too high"
   （ハードルが高すぎる）
```

**注意**: "Standard" は通常**複数形 "standards"** で使う

## 10. 「〜に盲目になる」

先生が教えてくれた重要表現：

```
"If you're in love, you tend to be blind about everything about your partner"
（恋をすると、パートナーの全てに盲目になりがち）
```

### 関連表現
| 表現 | 意味 |
|------|------|
| Love is blind | 恋は盲目 |
| Turn a blind eye to | 〜を見て見ぬふりをする |
| Be blinded by love | 愛に目がくらむ |
| Rose-colored glasses | バラ色の眼鏡（楽観的に見る） |

# トピック別：結婚・同棲・離婚の表現集

## 結婚について

### 結婚の意義を語る
| 表現 | 意味 |
|------|------|
| Marriage is a commitment | 結婚はコミットメントだ |
| Settle down with someone | 誰かと落ち着く |
| Tie the knot | 結婚する（イディオム） |
| Walk down the aisle | 結婚式をあげる |
| Pop the question | プロポーズする |

### 結婚のメリットを語る
```
✅ "You have someone to share your problems with"
   （問題を共有できる相手がいる）

✅ "You have a partner for life"
   （生涯のパートナーがいる）

✅ "You can build a family together"
   （一緒に家族を築ける）

✅ "You have emotional support"
   （精神的なサポートがある）
```

---

## 同棲について

### 同棲を勧める表現
```
✅ "You should live together before marriage"
   （結婚前に一緒に住むべき）

✅ "It's important to know if you're compatible"
   （相性が合うか知ることが大切）

✅ "You'll see their true colors"
   （本当の姿が見える）

✅ "You'll know their habits"
   （習慣がわかる）
```

### 相性に関する表現
| 表現 | 意味 |
|------|------|
| compatible | 相性が良い |
| incompatible | 相性が悪い |
| get along well | うまくやっていく |
| clash | ぶつかる |
| compromise | 妥協する |

---

## 離婚について

### 離婚の話題で使う表現
| 表現 | 意味 |
|------|------|
| file for divorce | 離婚を申請する |
| get a divorce | 離婚する |
| split up | 別れる |
| irreconcilable differences | 調停不能な相違 |
| custody of children | 子供の親権 |

### 離婚率上昇の理由を語る
```
✅ "Social media makes it easy to compare partners"
   （SNSでパートナーを比較しやすくなった）

✅ "Divorce is more socially acceptable now"
   （離婚が社会的に受け入れられるようになった）

✅ "People have higher expectations"
   （人々の期待値が高くなった）

✅ "Financial independence makes divorce easier"
   （経済的自立が離婚を容易にした）
```

### 独身のメリットを語る

   （許可を求める必要がない）

✅ "You can focus on your career"

# 今回学んだ重要構文

## 1. "tend to" の使い方

```
✅ "Women tend to overthink"
   （女性は考えすぎる傾向がある）

✅ "People tend to compare on social media"
   （人々はSNSで比較しがちだ）

✅ "Single people tend to work harder"
   （独身の人は働きすぎる傾向がある）
```

## 2. "in terms of" の使い方

```
✅ "In terms of financial problems..."
   （金銭的な問題という点では...）

✅ "In terms of compatibility..."
   （相性という点では...）
```

## 3. "end up" の使い方

```
✅ "It's gonna end up with a fight"
   （結局喧嘩になる）

✅ "Many marriages end up in divorce"
   （多くの結婚が離婚に終わる）
```

# 練習問題

2. Many couples _______ together before marriage. (live / leave)
3. The divorce _______ in Japan is about 30%. (rate / percentage)
4. Marriage is a life-_______ event. (changing / change)
5. Standards _______ too high these days. (is / are)

2. live
3. rate
4. changing
5. are

## 改善ポイント

| 間違えやすいペア | 対策 |
|-----------------|------|
| Married / Money | Rをはっきり発音 |
| Live / Leave | 短い「イ」vs 長い「イー」 |

### 文法：複数形に注意
- Standard → **Standards**
- Benefit → **Benefits**

### 語彙：信頼の表現をマスター
