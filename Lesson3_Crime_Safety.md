# 今回の重要な修正ポイント

## 1. 超重要！犯罪関連の語彙（音声認識の誤変換）

今回のレッスンで音声認識が誤変換した重要単語：

| 誤認識 | 正しい単語 | 意味 | 発音のコツ |
|--------|-----------|------|-----------|
| Rivalry | **Robbery** | 強盗 | /ˈrɒbəri/「ロバリー」 |
| Mother | **Murder** | 殺人 | /ˈmɜːrdər/「マーダー」 |
| Late writer | **The rate** | 割合 | /ðə reɪt/ |
| Climate | **Crime rate** | 犯罪率 | 「クライム・レイト」 |

```
⚠️ 特に注意！

❌ "Mother doesn't happen in Japan"
   → お母さんが日本で起こらない？？

✅ "Murder doesn't happen often in Japan"
   → 殺人は日本ではあまり起こらない
```

## 2.「いつも安全」の表現

### Original:
```
❌ "It's all time safety"
❌ "All time already a safety"
```

### Correction:
```
✅ "It's safe all the time"
   （いつも安全です）

✅ "It's safe around the clock"
   （24時間安全です）

✅ "It's safe day and night"
   （昼も夜も安全です）
```

### 解説
- "safety" は**名詞**、"safe" は**形容詞**
- 「安全だ」と言いたい時は **"It's safe"** を使う
- "all the time" は「いつも」の意味で、文末に置く

## 3.「〜に慣れていない」の表現

### Original:
```
❌ "I'm not used to feeling"
❌ "I'm not used to cloud transportation"
```

### Correction:
```
✅ "I'm not used to it"
   （それに慣れていない）

✅ "I'm not used to local transportation"
   （現地の交通機関に慣れていない）

✅ "I'm not used to riding motorcycles"
   （バイクに乗るのに慣れていない）
```

### "be used to" の構文
```
be used to + 名詞/動名詞（〜に慣れている）

✅ I'm used to working long hours.
   （長時間労働に慣れている）

✅ I'm not used to spicy food.
   （辛い食べ物に慣れていない）
```

**注意**: "used to + 動詞の原形" とは違う！
```
I used to live in Tokyo. = 昔東京に住んでいた（過去の習慣）
I'm used to living in Tokyo. = 東京に住むことに慣れている（現在の状態）
```

## 4.「〜しない方がいい」の表現

### Original:
```
❌ "I hadn't better to do ignore"
```

### Correction:
```
✅ "I had better not react"
   （反応しない方がいい）

✅ "It's better to just ignore them"
   （無視した方がいい）

✅ "I should just ignore them"
   （無視すべきだ）
```

### "had better" の構文
```
had better + 動詞の原形（〜した方がいい）
had better not + 動詞の原形（〜しない方がいい）

✅ You had better leave now.
   （今すぐ出発した方がいい）

✅ You had better not go there alone.
   （一人でそこに行かない方がいい）
```

**注意**: "had better to" は間違い！"to" は不要

# 今回学んだ重要構文

## 1. "be used to" vs "used to"

| 構文 | 意味 | 例 |
|------|------|-----|
| be used to + 名詞/動名詞 | 〜に慣れている | I'm used to hot weather. |
| used to + 動詞原形 | かつて〜した | I used to live in Tokyo. |

## 2. "had better (not)"

```
had better + 動詞原形 = 〜した方がいい
had better not + 動詞原形 = 〜しない方がいい

❌ had better to do
✅ had better do
```

## 3. 現在完了形で経験を語る

```
✅ Have you ever been to the Philippines?
✅ I have traveled to 5 countries.
✅ I have never experienced that.
```
