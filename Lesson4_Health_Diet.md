# 今回の重要な修正ポイント

## 1. 形容詞と名詞の混同

### Original:
```
❌ "to keep it health"
❌ "to keep healthy"（目的語なしで使う場合は正しい）
```

### Correction:
```
✅ "to keep it healthy"
   （それを健康に保つ）

✅ "to keep myself healthy"
   （自分を健康に保つ）

✅ "to stay healthy"
   （健康でいる）
```

### 解説
- "health" は**名詞**（健康）
- "healthy" は**形容詞**（健康な）
- 「〜を健康に保つ」= keep + 目的語 + healthy

```
❌ Bad: I want to keep my body health.
✅ Good: I want to keep my body healthy.
✅ Good: I want to stay in good health.
```

---

## 2. 頻度の表現「週に〜回」

### Original:
```
❌ "Twice, third a week"
❌ "twice a thrice a week"
```

### Correction:
```
✅ "Twice or three times a week"
   （週に2〜3回）

✅ "Two to three times a week"
   （週に2〜3回）

✅ "Twice or thrice a week"
   （週に2〜3回）※thriceはやや古風
```

### 頻度の表現まとめ

| 回数 | 表現 | 注意 |
|------|------|------|
| 1回 | once | ※one timeとは言わない |
| 2回 | twice | ※two timesとは言わない |
| 3回 | three times / thrice | thriceは古風 |
| 4回以上 | four times, five times... | |

```
✅ I go to the gym twice a week.
✅ I work out three times a week.
✅ I exercise every other day.（一日おきに）
```

---

## 3. 「近い」の前置詞

### Original:
```
❌ "My gym is very near from my office"
```

### Correction:
```
✅ "My gym is very near my office"
   （ジムはオフィスのすぐ近く）

✅ "My gym is close to my office"
   （ジムはオフィスの近く）
```

### 解説
- **near** は前置詞なので、直接目的語を取る（fromは不要）
- **close** は形容詞なので、**to** が必要

```
❌ Bad: near from ~
✅ Good: near ~
✅ Good: close to ~
```

---

## 4.「〜の後に」の表現

### Original:
```
❌ "after going to office, after I go to office"
```

### Correction:
```
✅ "after work"
   （仕事の後）

✅ "after I finish work"
   （仕事が終わった後）

✅ "after leaving the office"
   （オフィスを出た後）
```

### 解説
"go to office" より "go to the office" または "go to work" が自然。

```
❌ Bad: I go to gym after going to office.
✅ Good: I go to the gym after work.
✅ Good: I hit the gym after leaving the office.
```

## 5. 可算名詞と不可算名詞

### Original:
```
❌ "I like eating fruits. Fruits is very healthy"
```

### Correction:
```
✅ "I like eating fruit. Fruit is very healthy."
   （果物を食べるのが好き。果物は健康的だ）

✅ "I like various fruits."
   （いろいろな種類の果物が好き）
```

### 解説
- **Fruit** は通常**不可算名詞**として使う（全体として）
- 種類を強調する時だけ **fruits** を使う

| 使い方 | 例 |
|--------|-----|
| 一般的な話 | Fruit is good for you. |
| 種類を強調 | Tropical fruits are delicious. |

同様の例：
```
❌ Bad: I eat many vegetables. Vegetables is healthy.
✅ Good: I eat a lot of vegetables. They're healthy.
```

## 6.「カロリー」の正しい表現

### Original:
```
❌ "small curry"（音声認識の誤変換）
❌ "calorie made"
```

### Correction:
```
✅ "low calorie" / "low in calories"
   （低カロリー）

✅ "high in fiber"
   （食物繊維が豊富）

✅ "made with whole grains"
   （全粒穀物で作られた）
```

### 健康食品を説明する表現

| 表現 | 意味 |
|------|------|
| low calorie / low in calories | 低カロリー |
| high in protein | タンパク質が豊富 |
| rich in vitamins | ビタミンが豊富 |
| high in fiber | 食物繊維が豊富 |
| sugar-free | 砂糖不使用 |
| organic | オーガニック |

## 7.「つまらない」の使い分け

### Original:
```
❌ "it's boring, boring, boring taste"
```

### Correction:
```
✅ "It's bland"
   （味気ない、薄味）

✅ "It tastes boring"
   （味がつまらない）

✅ "It's not enjoyable to drink"
   （飲むのが楽しくない）
```

### 解説
- **boring** は「退屈な」（活動や状況に使う）
- **bland** は「味気ない、風味がない」（食べ物に使う）

```
❌ Bad: This food is boring.（食べ物が退屈？）
✅ Good: This food is bland.（この食べ物は味気ない）
✅ Good: This food tastes boring.（この食べ物は味が単調）
```

## 8. 現在進行形の省略

### Original:
```
❌ "I working today"
```

### Correction:
```
✅ "I'm working today"
   （今日は仕事です）

✅ "I have work today"
   （今日は仕事があります）
```

### 解説
進行形では **be動詞**（am/is/are）を省略してはいけない。

```
❌ Bad: I working from home.
✅ Good: I'm working from home.

❌ Bad: He coming later.
✅ Good: He's coming later.
```

## 9. メンタルヘルスの表現

### Original:
```
❌ "it doesn't affect to my mental effectively"
❌ "doesn't my mental recover"
```

### Correction:
```
✅ "It doesn't help my mental health"
   （精神的な健康に役立たない）

✅ "It doesn't improve my mood"
   （気分を良くしない）

✅ "It's not satisfying mentally"
   （精神的に満足しない）
```

### 解説
- **mental** は形容詞なので、単独で名詞として使わない
- **mental health**（精神的健康）を使う

```
❌ Bad: It affects my mental.
✅ Good: It affects my mental health.
✅ Good: It affects me mentally.
```

```
            （体重を管理する）
```

## ジム・運動について

### 運動の頻度を説明する
```
✅ "I work out twice a week"
   （週に2回運動する）

✅ "I've been hitting the gym more often lately"
   （最近よくジムに通っている）

✅ "I try to exercise regularly"
   （定期的に運動するようにしている）

✅ "I go to the gym every other day"
   （一日おきにジムに行く）
```

### 運動の目的を説明する
```
✅ "I exercise to stay healthy"
   （健康を維持するために運動する）

✅ "I work out to build muscle"
   （筋肉をつけるために運動する）

✅ "I go to the gym to warm up my body"
   （体を温めるためにジムに行く）

✅ "I exercise to relieve stress"
   （ストレス解消のために運動する）
```

### 運動時間を説明する
```
✅ "I spend about 30 minutes to an hour at the gym"
   （ジムで30分から1時間ぐらい過ごす）

✅ "My workouts usually last an hour"
   （トレーニングは普通1時間くらい）

✅ "I do a quick 30-minute workout"
   （30分の軽いトレーニングをする）
```

# 今回学んだ重要構文

## 1. keep + 目的語 + 形容詞

```
keep + 目的語 + 形容詞 = 〜を...の状態に保つ

✅ Keep yourself healthy.（健康でいなさい）
✅ Keep the room clean.（部屋をきれいに保って）
✅ Keep your body warm.（体を温かく保って）
```

## 2. 頻度の表現

```
once a week = 週に1回
twice a week = 週に2回
three times a week = 週に3回
every other day = 一日おきに
every day = 毎日
```

## 3. be into ~ / get into ~

```
be into ~ = 〜にハマっている
get into ~ = 〜にハマり始める

✅ I'm really into fitness.
   （フィットネスにすごくハマっている）

✅ I got into working out last year.
   （去年、運動にハマり始めた）
```
