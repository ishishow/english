# 今回の重要な修正ポイント

## 1. "percentage rate" → 重複表現

### Original:
```
❌ "the percentage rate is about 30%"
```

### Correction:
```
✅ "The divorce rate is about 30%"
   （離婚率は約30%）

✅ "The percentage is about 30%"
   （割合は約30%）

✅ "About 30% of marriages end in divorce"
   （結婚の約30%が離婚に終わる）
```

### 解説
- **rate** と **percentage** は似た意味なので、両方使うと重複
- どちらか一方を使う

---

## 2. "have married to divorce" → 不自然な表現

### Original:
```
❌ "Some couples have married to divorce"
```

### Correction:
```
✅ "Some couples end up getting divorced"
   （離婚することになるカップルもいる）

✅ "Some couples eventually divorce"
   （最終的に離婚するカップルもいる）

✅ "Some marriages end in divorce"
   （離婚で終わる結婚もある）
```

### 解説
- "have married to divorce" は文法的に不自然
- "end up ~ing" = 結局〜することになる

---

## 3. "I had" → "I heard"（音声認識の誤変換）

### Original:
```
❌ "I had the percentage rate is about 30%"
```

### Correction:
```
✅ "I heard that the divorce rate is about 30%"
   （離婚率は約30%だと聞いた）

✅ "I've heard that..."
   （〜だと聞いたことがある）
```

### 解説
- 音声認識で "heard" が "had" に誤変換されやすい
- 情報源を示す時は "I heard that..." を使う

---

## 4. "disappear" → "disapprove"（音声認識の誤変換）

### Original:
```
❌ "Some Japanese people may disappear with those people"
   （日本人がそれらの人々と消える？）
```

### Correction:
```
✅ "Some Japanese people may disapprove of those people"
   （離婚した人を良く思わない日本人もいる）
```

### 解説
- **disappear** = 消える
- **disapprove** = 不承認する、良く思わない
- 発音が似ているので注意！

| 単語 | 発音 | 意味 |
|------|------|------|
| disappear | /ˌdɪsəˈpɪr/ | 消える |
| disapprove | /ˌdɪsəˈpruːv/ | 不承認する |

---

## 5. "disapprove" の使い方

### Original:
```
❌ "I don't disapprove those people"
```

### Correction:
```
✅ "I don't disapprove of those people"
   （その人たちを悪く思わない）

✅ "I don't look down on divorced people"
   （離婚した人を見下さない）
```

### 解説
- **disapprove of** + 名詞（〜を良く思わない）
- **of** を忘れないこと！

```
❌ Bad: I disapprove his decision.
✅ Good: I disapprove of his decision.
```

---

## 6. 語順の問題「something + 形容詞」

### Original:
```
❌ "They have negative something"
```

### Correction:
```
✅ "They have something negative"
   （何かネガティブなものを持っている）

✅ "There's something wrong with them"
   （彼らには何か問題がある）
```

### 解説
- **something/anything/nothing + 形容詞** の語順
- 形容詞は後ろに置く

```
✅ something interesting（面白いこと）
✅ nothing special（特に何もない）
✅ anything wrong（何か問題）
```

---

## 7. "it will be news" → より自然な表現

### Original:
```
❌ "It will be news and broadcasting"
```

### Correction:
```
✅ "It will be all over the news"
   （ニュースで大きく取り上げられる）

✅ "It will be broadcast everywhere"
   （あちこちで放送される）

✅ "It will make headlines"
   （見出しになる）
```

---

## 8. "I may think" → "I might think"

### Original:
```
❌ "I may think they have negative something"
```

### Correction:
```
✅ "I might think they have something negative"
   （彼らに何かネガティブなものがあると思うかもしれない）

✅ "I tend to think..."
   （〜と思いがちだ）
```

### 解説
- **may** と **might** の違い：
  - **may** = 〜かもしれない（やや確信度高め）
  - **might** = 〜かもしれない（控えめ、仮定的）

自分の偏見や無意識の考えを認める時は **might** の方が適切

---

## 9. "involved" の意味と使い方

### Lesson中の質問:
```
"both people involved" = 関係している両者
```

### 解説
- **involved** = 関係している、巻き込まれている
- **be involved in** = 〜に関わっている

```
✅ "Everyone involved was happy"
   （関係者全員が幸せだった）

✅ "The people involved in the accident"
   （事故に巻き込まれた人々）

✅ "I don't want to get involved"
   （巻き込まれたくない）
```

---

## 10. "prejudice" の意味と使い方

### Lesson中の説明:
```
Prejudice = 偏見、先入観
unfair opinion or feeling about a person or group
without enough knowledge or experience
```

### 関連語彙

| 単語 | 意味 | 例 |
|------|------|-----|
| prejudice | 偏見 | racial prejudice（人種的偏見） |
| bias | バイアス、偏り | unconscious bias（無意識の偏見） |
| stereotype | 固定観念 | gender stereotype（性別の固定観念） |
| discrimination | 差別 | workplace discrimination（職場差別） |

```
✅ "We should avoid prejudice"
   （偏見を避けるべき）

✅ "Don't judge people based on stereotypes"
   （固定観念で人を判断しないで）
```

# トピック別：離婚・ジェンダー・偏見の表現集

## 離婚について話す

### 離婚率・統計
```
✅ "The divorce rate in Japan is about 30%"
   （日本の離婚率は約30%）

✅ "About one in three marriages ends in divorce"
   （3組に1組が離婚する）

✅ "The divorce rate is increasing"
   （離婚率は増加している）
```

### 離婚に対する意見
```
✅ "Divorce can be a positive decision in some cases"
   （離婚が前向きな決断になる場合もある）

✅ "It depends on the situation"
   （状況による）

✅ "When a marriage becomes unhealthy, divorce may be necessary"
   （結婚が不健全になったら、離婚が必要かもしれない）
```

### 離婚の影響
```
✅ "It's especially hard when there are children involved"
   （特に子供がいる場合は難しい）

✅ "Divorce can bring relief and a fresh start"
   （離婚は安心と新しいスタートをもたらすこともある）

✅ "It's a tough decision"
   （難しい決断だ）
```

## ジェンダーロールについて

### 伝統的な役割
```
✅ "Traditionally, husbands work outside and wives stay home"
   （伝統的に、夫は外で働き、妻は家にいる）

✅ "The husband is expected to be the breadwinner"
   （夫は稼ぎ手であることが期待される）

✅ "The wife is expected to do housework"
   （妻は家事をすることが期待される）
```

### 現代の変化
```
✅ "Gender roles are changing"
   （ジェンダーロールは変化している）

✅ "More couples share household responsibilities"
   （家事を分担するカップルが増えている）

✅ "Both partners can work and share childcare"
   （両方が働き、育児を分担できる）
```

## 偏見について

### 偏見を認める
```
✅ "I might have some unconscious bias"
   （無意識の偏見があるかもしれない）

✅ "I tend to think negatively about..."
   （〜についてネガティブに考えがち）

✅ "I try not to judge, but..."
   （判断しないようにしているが…）
```

### 偏見に反対する
```
✅ "We shouldn't judge divorced people"
   （離婚した人を判断すべきではない）

✅ "It's their personal decision"
   （それは個人の決断だ）

✅ "Everyone has their own reasons"
   （みんなそれぞれ理由がある）
```
