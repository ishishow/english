# 今回の重要な修正ポイント

## 1. 日本文化を英語で説明する

### おみくじ（Omikuji）

```
✅ "Omikuji is a fortune slip"
   （おみくじは運勢を占う紙）

✅ "At a shrine, you pay 100 yen and draw a paper
    that tells your fortune for the year"
   （神社で100円払って、その年の運勢が書かれた紙を引く）

✅ "It covers various aspects like love, career, health, etc."
   （恋愛、仕事、健康など様々な面について書かれている）
```

### お年玉（Otoshidama）

```
✅ "Otoshidama is New Year's money that adults give to children"
   （お年玉は大人が子供にあげるお正月のお金）

✅ "Parents, grandparents, and relatives give money to children
    during New Year"
   （両親、祖父母、親戚が正月に子供にお金をあげる）

✅ "Usually, children under 20 can receive otoshidama"
   （通常、20歳未満の子供がお年玉をもらえる）
```

## 2. "cloud" vs "crowd"（音声認識の誤変換）

### Original:
```
❌ "I can stand cloud"
```

### Correction:
```
✅ "I can handle crowds"
   （混雑を我慢できる）

✅ "I can tolerate crowded places"
   （混雑した場所を耐えられる）

✅ "I don't mind crowds"
   （混雑は気にならない）
```

### 解説

| 単語 | 発音 | 意味 |
|------|------|------|
| cloud | /klaʊd/ | 雲 |
| crowd | /kraʊd/ | 群衆、混雑 |

```
⚠️ 発音の違いに注意！
cloud = クラウド（cl-）
crowd = クラウド（cr-）
Rの音が入る！
```

## 3. "stand" vs "handle/tolerate"

### Original:
```
❌ "I can stand crowd"（不自然）
```

### Correction:
```
✅ "I can handle crowds"
   （混雑に対処できる）

✅ "I can tolerate crowds"
   （混雑を我慢できる）

✅ "I can put up with crowds"
   （混雑に耐えられる）
```

### 解説
- **stand** は「立つ」が基本意味。「我慢する」の意味もあるが、少しカジュアル
- **handle** は「対処する」というニュアンス
- **tolerate** は「我慢する」の正式な表現

## 8. 時制の使い分け：現在完了 vs 過去形

### "we've been to Taiwan"（正しい！）

```
✅ "Last year, we went to Taiwan"
   （去年、台湾に行った）← 過去形：特定の時点

✅ "We've been to Taiwan before"
   （台湾に行ったことがある）← 現在完了：経験

✅ "We celebrated New Year's Eve in Taiwan"
   （台湾で大晦日を祝った）← 過去形：出来事
```

### Shoさんの発話：
```
"The last year we went to we've been to Taiwan"
```

### 解説
- 途中で言い直したが、**過去形 "went"** が正しい
- 特定の時点（去年）を示しているため

## 9. "graduate from this culture"（ユニークな表現）

### Original:
```
"graduate this culture"
```

### Correction:
```
✅ "I've outgrown that tradition"
   （その伝統を卒業した）

✅ "I'm too old for that now"
   （もうその年齢じゃない）

✅ "That tradition doesn't apply to me anymore"
   （もうその伝統は自分には当てはまらない）
```

### 解説
"graduate from" は学校を卒業する時に使う。
文化や伝統を「卒業」するには "outgrow" や "no longer apply" を使う。

## 10. 過去形 vs 現在形の混乱

### Original:
```
❌ "I worked at home today"
   （今日在宅で働いた）← 今日のことなのに過去形
```

### Correction:
```
✅ "I'm working from home today"
   （今日は在宅勤務です）← 進行形

✅ "I work from home today"
   （今日は在宅で働く）← 現在形（予定）

✅ "Today is a work-from-home day for me"
   （今日は私の在宅勤務日）
```

### 解説
「今日」の出来事について話すときは：
- 進行形：今まさにしていること
- 現在形：予定・習慣
- 過去形は使わない（まだ終わっていないため）
