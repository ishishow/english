# 今回の重要な修正ポイント

## 1. "it isn't my routine" → より自然な表現

### Original:
```
❌ "It isn't my routine"
   （TikTokを使わない理由として）
```

### Correction:
```
✅ "It's not part of my routine"
   （それは私のルーティンの一部ではない）

✅ "It's not something I usually do"
   （普段やることではない）

✅ "I'm not in the habit of using it"
   （使う習慣がない）
```

### 解説
- "my routine" だけでは少し不自然
- "part of my routine" で「ルーティンの一部」という表現に
- または「習慣がない」という別の言い方も可能

## 2. "I'm not good at posting" → 適切な表現

### Original:
```
❌ "I'm not good at posting picture"
```

### Correction:
```
✅ "I'm not good at posting pictures"
   （写真を投稿するのが得意ではない）

✅ "I don't really post pictures"
   （あまり写真を投稿しない）

✅ "I prefer browsing to posting"
   （投稿するより見る方が好き）
```

### 解説
- **picture** は可算名詞なので複数形 **pictures** に
- 「得意ではない」より「しない」の方が自然な場合も

---

## 3. "just looking" → より自然な表現

### Original:
```
❌ "I'm not good at posting picture and just looking"
```

### Correction:
```
✅ "I mostly just browse"
   （ほとんど見るだけ）

✅ "I just scroll through my feed"
   （フィードをスクロールするだけ）

✅ "I just look at other people's posts"
   （他の人の投稿を見るだけ）
```

### 解説
- **browse** = 閲覧する、見て回る
- **scroll** = スクロールする（SNS特有の表現）
- **feed** = フィード（SNSで流れてくる投稿一覧）

## 8. "not familiar either with me" → 語順の問題

### Original:
```
❌ "not familiar area or familiar either with me"
```

### Correction:
```
✅ "areas that are unfamiliar to me"
   （自分にとって馴染みのない場所）

✅ "places I've never been to"
   （行ったことのない場所）

✅ "countries I'm not familiar with"
   （馴染みのない国）
```

### 解説
- **be familiar with** = 〜に馴染みがある
- **be unfamiliar to** = 〜にとって馴染みがない

```
✅ I'm familiar with Tokyo.（東京には馴染みがある）
✅ Africa is unfamiliar to me.（アフリカは馴染みがない）
```

## SNSの良い点・悪い点

### 良い点 (Advantages)
```
✅ "It's easy to stay connected with friends"
   （友達と繋がりやすい）

✅ "We can access a lot of information quickly"
   （多くの情報にすぐアクセスできる）

✅ "It's useful for work and research"
   （仕事やリサーチに便利）

✅ "We can learn about other cultures"
   （他の文化について学べる）
```

### 悪い点 (Disadvantages)
```
✅ "It can be a waste of time"

   （中毒性がある）

✅ "Not everything online is reliable"
   （オンラインのすべてが信頼できるわけではない）

✅ "It affects my sleep"
   （睡眠に影響する）
```
