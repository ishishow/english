# はじめに - この教材について

## Shoさんの英語力の特徴

あなたの英語は**「言いたいことを伝える力（Communicative Competence）」**が非常に高いです。
文法ミスがあっても、文脈や補足説明で相手に理解させています。これは素晴らしいスキルです。

## この教材の目的

次のステージは**「文脈に合った単語のチョイス（Register）」**です。

| シーン | 目指すスタイル |
|--------|---------------|
| 歴史の話 | 誤解を生まない慎重な言葉 |
| 趣味の話 | カジュアルで情熱的な言葉 |
| 仕事の話 | プロフェッショナルな表現 |
| 日常の話 | シンプルな文法 |

## 学習の進め方

1. **各章を順番に読む** - 解説を理解する
2. **例文を音読する** - 口に馴染ませる
3. **Bad/Good例を比較する** - 違いを体感する
4. **Chapter 8の練習問題で確認** - 定着度チェック

# Chapter 1: 品詞・基本文法編

## 1.1 名詞と動詞の混同

### Success（名詞） vs Succeed（動詞）

これは日本人学習者が非常によく間違えるポイントです。

| 品詞 | 単語 | 意味 |
|------|------|------|
| 名詞 | Success | 成功 |
| 動詞 | Succeed | 成功する |

```
❌ Bad: If I success to go to bed at 12...
✅ Good: If I succeed in going to bed at 12...
✅ Better: If I manage to get to bed by 12...
```

**ポイント**: 日常会話で「なんとか〜する」は **manage to** が自然！

### その他の名詞/動詞ペア

| 名詞 | 動詞 | 例文 |
|------|------|------|
| advice | advise | Can you advise me? |
| choice | choose | I chose this option. |
| difference | differ | Our opinions differ. |

---

## 1.2 前置詞の正しい使い方

### 基本動詞 + 前置詞セット

これらを**塊で覚える**と、もっとスムーズになります。

| 動詞 | 前置詞 | 例文 |
|------|--------|------|
| work | as | I work **as** an engineer. |
| live | in | I live **in** a share house. |
| go | to | I went **to** Japan. |
| listen | to | Listen **to** me. |
| belong | to | I belong **to** this team. |

```
❌ Bad: Most people work full time worker.
✅ Good: Most people work as full-time employees.

❌ Bad: I came back Japan.
✅ Good: I came back to Japan.
```

---

## 1.3 比較の構文

### Compared to ~ を文頭に置くテクニック

日本語の「〜と比べると」を英語にするとき、最もスムーズな方法：

```
❌ Bad: It's not difficult to compare with American university.
✅ Good: Compared to American universities, it's not that difficult.
```

**構文パターン**:
```
Compared to [A], [B] is [形容詞].
（Aと比べると、Bは〜だ）
```

**例文**:
- Compared to other countries, Japanese athletes are famous for Judo.
- Compared to Tokyo, this city is very peaceful.

---

## 1.4 時制の使い分け

### 過去形 vs 現在完了形

| 時制 | 使う場面 | 例 |
|------|---------|-----|
| 過去形 | 過去の一点の出来事 | I played baseball in high school. |
| 現在完了形 | 過去から現在に続く状態 | I haven't decided yet. |

```
❌ Bad: I actually decided not.
✅ Good: I actually haven't decided on a specific topic yet.
```

### 過去完了形は本当に必要？

過去完了（had + pp）は「過去のある時点より**さらに前**」を強調する時だけ使います。

```
❌ Overuse: When I was a student, I had played baseball for a long time.
✅ Simple: I played baseball for years when I was a student.
✅ Better: I used to play baseball.
```

**used to**: 「昔はやっていた（今はやっていない）」というニュアンス

# Chapter 2: 動詞マスター編

## 2.1 Win vs Beat - 超重要！

これは**必ず覚えるべき**使い分けです。

| 動詞 | 目的語 | 意味 |
|------|--------|------|
| **Win** | 試合/賞品 | 勝ち取る |
| **Beat** | 対戦相手 | 打ち負かす |

```
✅ Win the game (試合に勝つ)
✅ Win a prize (賞を勝ち取る)
✅ Win a gold medal (金メダルを獲得する)

✅ Beat my father (父に勝つ)
✅ Beat the other team (相手チームを倒す)
✅ Beat him (彼を負かす)
```

### よくある間違い

```
❌ Bad: I don't have mind to win my father.
        → 「父を賞品として獲得する」という不思議な意味に！

✅ Good: I don't think I can beat my father.
✅ Good: I don't intend to beat my father.
```

### 関連イディオム

| 表現 | 意味 |
|------|------|
| He's out of my league. | 彼は私とはレベルが違う（強すぎる） |
| He wipes the floor with me. | 彼は私を完膚なきまでに叩きのめす |

---

## 2.2 使役動詞 - Make / Have / Get / Let

### 基本の意味

| 動詞 | ニュアンス | 強制度 |
|------|-----------|--------|
| **Make** | 強制的に〜させる | ★★★★★ |
| **Have** | 依頼して〜してもらう | ★★★☆☆ |
| **Get** | 説得して〜させる | ★★★☆☆ |
| **Let** | 許可して〜させてあげる | ★☆☆☆☆ |

### 構文パターン

```
Make + 人 + 動詞の原形
Have + 人 + 動詞の原形
Get + 人 + to + 動詞
Let + 人 + 動詞の原形
```

### 実践例（お父さんとの関係で使える）

```
❌ Bad: My father changed my club soccer to baseball.
✅ Good: My father made me switch from soccer to baseball.
        （父が私にサッカーから野球に変えさせた）

✅ My father made me play golf.
   （父が私にゴルフをやらせた）

✅ My father let me quit the club.
   （父が私に部活を辞めさせてくれた）
```

### 美容院の表現（Get/Have + 物 + 過去分詞）

自分でやるのではなく、**誰かにやってもらう**場合：

```
❌ Bad: I will make my hair cut.
✅ Good: I'm going to get a haircut.
✅ Good: I'm going to have my hair cut.
```

**構文**: Get/Have + 物 + 過去分詞

---

## 2.3 重要な動詞の使い分け

### Hear vs Listen

| 動詞 | 意味 | 例 |
|------|------|-----|
| Hear | （自然に）聞こえる | I heard that... |
| Listen | （意識的に）聴く | Listen to me. |

```
❌ Bad: I had the number of Korean people increased.
✅ Good: I heard that the number of Korean students has increased.
```

### Study vs Start（趣味を始める）

```
❌ Bad: I studied playing golf this year.
✅ Good: I started playing golf this year.
✅ Better: I took up golf this year.
```

**Take up**: 新しい趣味や活動を「始める」

---

## 2.4 Resist / Manage / Commute

### Resist（抵抗する、誘惑に勝つ）

```
❌ Bad: I can't beat my Netflix.
✅ Good: I can't resist the temptation of Netflix.
✅ Good: Netflix is too distracting.
```

### Manage to（なんとか〜する）

```
✅ If I manage to get to bed by 12, I'm happy.
✅ I managed to finish the project on time.
```

### Commute（通勤する）

```
❌ Bad: I don't need to go to my office.
✅ Good: I don't need to commute.
✅ Better: Since I work remotely, I don't need to commute.
```

# Chapter 3: 形容詞・分詞の使い分け

## 3.1 Scared vs Scary（超重要！）

### ルール

| パターン | 対象 | 例 |
|----------|------|-----|
| **-ed形** | 人の感情 | I am scared. (私は怖がっている) |
| **-ing形/形容詞** | 物事の性質 | It is scary. (それは怖い) |

```
❌ Bad: I was scary about AI.
        → 「私は（他人にとって）怖い人だった」という意味に！

✅ Good: I'm scared of AI evolving.
✅ Good: I'm worried about AI taking over our jobs.
```

### 同じルールの他の形容詞ペア

| -ed (人の感情) | -ing (物の性質) |
|---------------|----------------|
| bored | boring |
| excited | exciting |
| interested | interesting |
| tired | tiring |
| confused | confusing |

```
❌ Bad: I am boring.
        → 「私はつまらない人間だ」

✅ Good: I am bored.
        → 「私は退屈している」
```

---

## 3.2 形容詞の選び方

### Tough vs Sensitive vs Heavy（話題について）

```
❌ Neutral: Discrimination is a tough topic.
✅ Better: Discrimination is a sensitive topic.
✅ Better: Discrimination is a heavy topic.
```

**sensitive**: 繊細で扱いづらい
**heavy**: 重い、深刻な

### Enthusiastic vs Hooked（趣味について）

```
❌ Formal: I am enthusiastic about golf.
          （面接のような硬い響き）

✅ Casual: I'm totally hooked on golf.
✅ Casual: I've really gotten into it recently.
✅ Casual: I'm obsessed with it.
```

---

## 3.3 注意が必要な形容詞

### Normal vs Understandable

歴史的な感情について話すとき：

```
⚠️ Risky: It's normal to feel resentment.
          （「それが正常な状態」と聞こえる可能性）

✅ Better: It's understandable that they still harbor resentment.
          （「無理もない、理解できる」）
```

### Civilized - 使用注意！

```
⚠️ Risky: We are already civilized now.
          （「昔は野蛮だった」というニュアンスを含む）

✅ Better: We are a modern, independent nation now.
✅ Better: Society has evolved since then.
```

## 5.5 差別・社会問題について

### 人間の本能を説明する

```
❌ Basic: We like our human like teams... want teammates...

✅ Advanced:
"It stems from human instinct."
（それは人間の本能に由来します）

"Humans instinctively want to belong to a group of like-minded people."
（人間は本能的に、似た考えの人々のグループに属したがります）

"This creates an 'Us vs. Them' mentality, which leads to discrimination."
（これが「我々 対 彼ら」という心理を生み、差別につながります）
```

### Stereotype / Prejudice / Discrimination の違い

| 概念 | 種類 | 意味 | 例 |
|------|------|------|-----|
| Stereotype | 認知 | 固定観念 | 「日本人は金持ちだ」 |
| Prejudice | 感情 | 偏見 | 「だから気に入らない」 |
| Discrimination | 行動 | 差別行為 | 「だからお金を要求する」 |

```
✅ They assume I'm rich because I'm Japanese.
   （彼らは私が日本人だから金持ちだと思い込んでいます）

✅ That's more of a stereotype than discrimination.
   （それは差別というよりステレオタイプですね）
```

# Quick Reference Card（クイックリファレンス）

## 最優先で覚えるべき5項目

### 1. Win vs Beat
```
Win = 試合・賞品を勝ち取る (Win the game, Win a medal)
Beat = 対戦相手を倒す (Beat my father, Beat the team)
```

### 2. Scared vs Scary
```
I am scared = 私は怖がっている（人の感情）
It is scary = それは怖い（物の性質）
```

### 3. 使役動詞（美容院で使う）
```
❌ I will make my hair cut.
✅ I'm going to get a haircut.
```

### 4. 比較の構文
```
❌ It's not difficult to compare with America.
✅ Compared to America, it's not difficult.
```

### 5. 発音注意（R vs L）
```
Lavish = 豪華な（Lから始まる）
Rubbish = ゴミ（Rから始まる）
```
