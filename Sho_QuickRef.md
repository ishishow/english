# 最優先 ★★★

## 使役構文

```
髪を切る     → get a haircut / have my hair cut
車を直す     → get my car fixed
写真を撮る   → get my photo taken
```

**絶対NG:** ~~make my hair cut~~

---

## 前置詞セット

```
work FOR + 会社    I work FOR Google.
work AS + 職種     I work AS an engineer.
work AT + 場所     I work AT the office.
work IN + 業界     I work IN IT.

live IN + 場所     I live IN Tokyo.
live WITH + 人     I live WITH friends.

go TO + 場所       I went TO Cebu.
come back TO       I came back TO Japan.
```

---

# 重要 ★★

## 時制シグナルワード

```
yet/already/ever/never → 現在完了形
  例: I haven't decided YET.

yesterday/last~/ago → 過去形
  例: I didn't go YESTERDAY.
```

---

## L/R 発音

```
L = 舌先を歯茎につける
R = 舌はどこにもつけない

要注意ペア:
lavish ≠ rubbish  (豪華な ≠ ゴミ)
light ≠ right     (光 ≠ 正しい)
lock ≠ rock       (鍵 ≠ 岩)
```

# 発展 ★

## 比較表現

```
Compared to ~, ...    〜と比べると
~ than in [場所]      [場所]より
not as ~ as ...       ...ほど〜ない
```

## 値段表現

```

  You can't beat that price.
```

