# SLIDE 8: L/R の発音

```
┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃                                                   ┃
┃  Shoさんの発言が誤認識された例                    ┃
┃  ─────────────────────────────                    ┃
┃                                                   ┃
┃  言いたかった: lavish lifestyle (豪華な生活)      ┃
┃  認識された:   rubbish lifestyle (ゴミみたいな)   ┃
┃                                                   ┃
┃  ───────────────────────────────────────────      ┃
┃  L = 舌先を歯茎につける                           ┃
┃  R = 舌はどこにもつけない                         ┃
┃                                                   ┃
┃  練習: "I want a LAVISH lifestyle,                ┃
┃         not a RUBBISH one."                       ┃
┃                                                   ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛
```

# 会話シミュレーション

## Scenario 1: 自己紹介

```
Teacher: Tell me about yourself.

You: I work FOR a tech company AS a software engineer.
     I can work remotely, so I'm considering becoming
     a digital nomad in the future.

Teacher: Where do you live?

You: I live IN a share house WITH about 50 residents.
     It's a massive community!

Teacher: What are your goals?

You: I'm looking to switch careers to an international
     company. English is a must-have skill for the
     positions I'm interested in.
```

## Scenario 2: 旅行の話

```
Teacher: Have you traveled abroad?

You: Yes! I went TO Cebu for a few months.
     I got a diving license there.

Teacher: How was the ocean?

You: The water was crystal clear! The visibility was
     amazing. It was worth every penny.

Teacher: Any problems?

You: Actually, a typhoon hit while I was there.
     The power went out for a few days.
     But the show must go on—I still took classes!
```

## Scenario 3: 美容室の話

```
Teacher: How much is a haircut in Japan?

You: It'll set you back about 4,000 yen.
     That's a bit steep, don't you think?

Teacher: How about in the Philippines?

You: Only 700 yen! That's a steal!
     You can't beat that price.
```

## Scenario 4: 価値観の話

```
Teacher: What kind of life do you want?

You: I want to live an unconventional lifestyle.
     I want to be unique and go off the beaten path.

Teacher: What does that mean for you?

You: I want to travel the world while working remotely.
     I want to live a LAVISH lifestyle—not a RUBBISH one!
     (L の発音注意！)
```
